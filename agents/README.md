# Agent Roles

Prompts and configs for the AI agents that facilitate ToastAgents events.

## Roles

- **Timer** - Tracks speaker time, announces warnings
- **Feedback** - Generates brief evaluations after each talk
- **Hot Seat** - Picks audience members, generates questions
- **Ah Counter** - Tracks filler words (um, ah, like, you know, so, basically, right?)
- **Recap** - Summarizes the night's key moments

## Setup

Each agent runs as a Claude session on a laptop connected to the venue display. See individual agent files for prompts and configuration.
