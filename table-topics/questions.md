# Table Topics Question Bank

Questions for the hot seat round. 60-120 second answers. No preparation.

## Workflows & Tools

- Describe your Claude Code setup in 60 seconds.
- What's one thing you automated this week that you probably shouldn't have?
- What's a workflow you saw on Twitter that actually worked?
- What's the most overengineered thing you've built with AI help?
- Walk us through your morning coding routine with AI.

## Wins & Failures

- What's the worst thing an AI agent ever did to your codebase?
- Tell us about a bug that Claude found that you never would have.
- What's the biggest time save you've gotten from AI tooling this month?
- Describe a moment where you realized AI had changed how you think about code.
- What's something you stopped doing manually because of AI?

## Opinions & Hot Takes

- Is vibe coding real engineering? You have 60 seconds.
- What's one AI workflow trend that's overhyped?
- Should junior developers use AI coding tools? Why or why not?
- What's one thing Claude Code is terrible at?
- Will we still write code manually in 5 years?

## Show & Tell

- What's the weirdest thing you've used an LLM for?
- Show us one file on your machine that proves you're an AI power user.
- What's your most used custom instruction or system prompt?
- What's a tool or MCP server you built that nobody knows about?
