# Contributing to ToastAgents

## Adding Your Chapter

Add a row to the Chapters table in README.md with:
- City
- Organizer (with link)
- Cadence (monthly, bi-weekly, etc.)
- Start date

## Adding Table Topics Questions

Add questions to `table-topics/questions.md`. Good questions are:
- Answerable in 60-120 seconds
- About real experiences, not hypotheticals
- Specific enough to trigger a story, broad enough for anyone to answer

## Adding Event Recaps

Create a file in `recaps/` named `YYYY-MM-DD-city.md`. Include:
- Theme of the night
- Number of attendees
- Lightning talk summaries (no transcripts - keep it brief)
- Best table topics moments
- What worked, what to change

## Agent Configs

If you've built or modified agent prompts for Timer, Feedback, Hot Seat, or Recap roles, add them to `agents/`.
