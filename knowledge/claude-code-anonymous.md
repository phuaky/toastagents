# Claude Code Anonymous

The origin format that inspired ToastAgents.

## What It Is

Claude Code Anonymous is a meetup format coined by Peter Steinberger (@steipete), creator of OpenClaw. Born from a conversation between Peter and Orta Therox where they realized Claude Code had completely changed how they build software - to the point where they couldn't stop thinking about it.

The name is a play on support group meetings (like Alcoholics Anonymous), reflecting how builders feel "addicted" to AI-assisted development.

## The Format

- Lightning talks, about 5 minutes each
- Opening format: "I was X when Claude Code Y"
- Examples: "I was impressed when Claude Code figured out a bug I'd been stuck on" or "I was disappointed when Claude Code deleted my production database"
- Not recorded. No livestream. Keeps it candid.
- Intro and talks kept to one hour
- 2-3 hours for socializing after
- The real goal is meeting like-minded people

## The Black Eye Club

Peter coined this term for the type of person most drawn to CCA: full-breadth developers with both technical and product capabilities who adopt AI-first building the fastest - and who usually have trouble sleeping because of it.

## Spread

People who couldn't make it to the first events were encouraged to start their own chapters. Events have been held in London, Vienna, Berlin, Cologne, San Francisco, Delft, and more.

## What ToastAgents Adds

CCA is deliberately unstructured - just talks and socializing. ToastAgents adds:
- Structured roles (handled by AI agents instead of humans)
- Table Topics / Hot Seat for audience participation
- Ah Counter for filler word tracking
- Event lifecycle tracking (plan, recap, retro)
- A queryable content database for organizers

The "I was X when Claude Code Y" opening format is preserved as core DNA.
