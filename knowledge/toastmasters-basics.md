# Toastmasters Basics for ToastAgents

Core Toastmasters concepts adapted for builder meetups.

## The Core Idea

Toastmasters exists because most people are bad at public speaking and the only way to get better is practice in a safe environment. ToastAgents applies this to builders who are bad at presenting technical work.

## Key Roles (Traditional → ToastAgents)

| Toastmasters | ToastAgents | Difference |
|-------------|-------------|------------|
| Toastmaster of the Evening | Host | Same role - runs the show |
| Timer | Timer Agent | AI replaces human volunteer |
| Evaluator | Feedback Agent | AI generates evaluations |
| Table Topics Master | Hot Seat Agent | AI picks people and generates questions |
| Ah Counter | Ah Counter Agent | AI tracks filler words |
| Grammarian | (Not used) | Replaced by Ah Counter |
| General Evaluator | Recap Agent | AI summarizes the night |

## Table Topics

The most transferable Toastmasters concept. Rules:
- Someone gets called up with zero preparation
- They get a question or prompt
- They speak for 1-2 minutes
- The audience evaluates (or just enjoys)

Why it works: It trains the muscle of thinking on your feet. Developers need this for standups, demos, investor pitches, and job interviews.

## The Evaluation Framework

Toastmasters evaluations follow CRC: Commend, Recommend, Commend.
1. Start with something specific that worked
2. Give one concrete improvement
3. End with encouragement

ToastAgents uses the same structure but the Feedback Agent delivers it.

## Speech Structure

The simplest effective structure for a 5-minute talk:
1. **Hook** (30 sec) - "I was [emotion] when Claude Code [thing]"
2. **Context** (60 sec) - Set the scene. What were you working on?
3. **Conflict** (90 sec) - What went wrong/right/weird?
4. **Resolution** (60 sec) - What happened? What did you learn?
5. **Takeaway** (30 sec) - One thing the audience can use

## Timing Discipline

Toastmasters is strict about time because it teaches a real skill: saying what matters and cutting what doesn't. ToastAgents keeps this discipline.

- Green: On time
- Yellow: 1 minute remaining
- Red: Time's up
- Over: You're done. Wrap it up.

## What ToastAgents Drops from Toastmasters

- **Pathways curriculum** - Too structured for a casual meetup
- **Grammarian role** - Not relevant for technical talks
- **Word of the Day obligation** - Optional, replaced by Term of the Day
- **Awards voting** - Optional, can add if the chapter wants it
- **Formal meeting structure** - Simplified to 3 segments: opening, talks, hot seat
